（3）采用命令将echo-nginx-module-0.60移至echo-nginx-module目录。
（4）采用命令进入目录nginx-1.10.3

cd nginx-1.10.3

（5）执行命令生成makefile，并且添加echo-nginx-module模块
（6）执行命令生成可执行的二进制文件

​ make && make install

（7）采用命令备份原来的nginx可执行的二进制文件
（8）采用命令将新编译的nginx可执行的二进制文件复制到nginx的安装目录sbin
（9）修改nginx的主配置文件
（10）采用命令重新启动nginx
（11）打开浏览器进行测试，如果成功看到如下表示成功：

This is an echo module.

vim index.html

F12–>network–>Headers–>ResponseHeaders

防盗链配置

（１）打开nginx的主配置的主配置文件，添加名为www.xp5.com和www.img.com的虚拟主机：
（２）在html/www.img.com中建立子目录img，同时上传一些图片到该目录

cd /usr/local/nginx/html

mkdir www.img.com

cd www.img.com

上传图片

（３）在网站www.img.com中建立测试页面index.html

vim index.html

（４）在网站www.xp5.com中盗用www.img.com中的图片，建立index.html，输入如下代码：

vim index.html

（５）打开浏览器，分别输入进行访问，发现www.xp5.com盗链成功
（６）修改www.img.com对应的配置文件，实现防盗链
（７）重新启动nginx

cd /usr/local/nginx/sbin ./nginx -s reload

（８）打开浏览器进行测试，如果发现看不到图片，则防盗链成功。
