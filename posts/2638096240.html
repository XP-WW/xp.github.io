Linux安装Redis

（1）使用wget命令下载redis-6.0.8.tar.gz

wget http://download.redis.io/releases/redis-6.0.8.tar.gz

（2）解压源码并编译安装
（3）启动 redis 服务
（4）打开另一终端，使用 redis-cli 和 redis 服务交互

PHP 使用 Redis

（1）PHP安装redis扩展
（2）修改php.ini文件

vi /usr/local/php/lib/php.ini

（3）安装完成后重启php-fpm,查看phpinfo信息，就能看到redis扩展。

service php-fpm restart

（4）测试Redis服务器：建立一个名为test.php的页面，并放至nginx或apache的相应站点目录。

vim test.php

（5）打开windows浏览器，输入服务器的http://IP/test.php，若看到如下输入，表示Redis服务器连接成功：

Redis PHP String(字符串) 实例

（1）新建立一名为use_string.php的页面，页面内容如下所示：

vim use_string.php

（2）打开windows浏览器，输入服务器的http://IP/use_string.php

Redis PHP List(列表) 实例

（1）新建立一名为use_list.php的页面，页面内容如下所示：

vim use_list.php

（2）打开windows浏览器，输入服务器的http://IP/use_list.php并记录下实验结果
